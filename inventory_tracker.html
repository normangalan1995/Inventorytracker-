üìä Export to Google Sheets

To create a working Google Sheets version of this tracker:

  1. Create a new Google Sheet
  2. Create separate tabs for each location (DDI, Drive-Up, Epiphany, Improv Shop)
  3. Add a "Master Schedule" tab showing all deliveries by day
  4. Include columns for: Item, Vendor, Orderer, Receiver, Delivery Day, Current Stock, Reorder Point, Last Order Date, Next Order Due
  5. Set up data validation for dropdown menus (personnel, vendors, etc.)
  6. Create conditional formatting for low stock alerts
  7. Add a dashboard tab with charts showing vendor distribution, delivery schedule, and key metrics

Recommendation: Start with the DDI tab as a template since it has the most detailed process documentation.
